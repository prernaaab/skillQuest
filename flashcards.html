Know Thy Basics

Explore the fundamentals of Python programming through these informative flashcards.

Variables

Variables store data values that can change during program execution.

Example: my_variable = 10

Data Types

Python supports several data types including int, float, str, and list.

Example: my_list = [1, 2, 3]

Lists

Lists are ordered collections that can hold items of different data types.

Example: my_list = [1, "apple", 3.14]

Tuples

Tuples are similar to lists but are immutable (cannot be changed).

Example: my_tuple = (1, 2, 3)

Dictionaries

Dictionaries store data in key-value pairs, allowing for easy data retrieval.

Example: my_dict = {"name": "John", "age": 30}

Loops

Loops are used to iterate over sequences such as lists or strings.

Example: for i in range(5): print(i)

Functions

Functions are reusable pieces of code that perform specific tasks.

Example: def my_function(): return "Hello"

Conditionals

Conditionals execute code based on whether a condition is true or false.

Example: if x > 10: print("x is greater than 10")

Modules

Modules are files that contain Python code, allowing code reuse.

Example: import math

File I/O

Python can read from and write to files using built-in functions.

Example: with open("file.txt", "r") as f: content = f.read()

Exceptions

Exceptions are used to handle errors gracefully in code.

Example: try: x = 1 / 0 except ZeroDivisionError: print("Error!")

Classes and Objects

Python supports Object-Oriented Programming (OOP) with classes and objects.

Example: class MyClass: pass

List Comprehension

A concise way to create lists based on existing lists.

Example: squares = [x**2 for x in range(10)]

Lambda Functions

Lambda functions are small anonymous functions defined using the lambda keyword.

Example: add = lambda x, y: x + y

List Methods

Common methods for manipulating lists include append, remove, and pop.

Example: my_list.append(4)

String Methods

String methods allow for manipulation of string objects.

Example: my_string.upper()

API Requests

Python can make HTTP requests to interact with APIs using libraries like requests.

Example: import requests
